Sobre o site

CarlãoDex é uma pokedex em forma de site que utiliza somente das técnologias web básicas em sua estrutura (HTML,CSS e JavaScript).

O site obtem os dados de todos os pokemons atravez da API PokeAPI

O site foi feito como forma de projeto para aperfeiçoar as habilidades de programação WEB do desenvolvedor, que pode ser visto na aba contato .

O site pode mostrar listar todos os pokemons disponíveis na API como também buscar um pokemon para ver seus tipos, sprites e status base.
